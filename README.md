The usage is rather simple, however it does require python and the library pygame to be installed.
Installing pygame merely requires you to input "pip install pygame" (excluding the quotation marks) into your terminal / command prompt.

There are a few sample midi's, although you can just as easily place a few new midi's within the program folder and it'll work just fine.
To start the program, execute Midi Player.py and enter the name of the midi you wish to play.
